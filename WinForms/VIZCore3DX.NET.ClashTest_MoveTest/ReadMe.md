# VIZCore3DX.NET.SectionBoxSize

## 개요
`VIZCore3DX.NET.ClashTest_MoveTest` 프로젝트는 VIZCore3DX.NET 라이브러리를 사용하여 3D 모델의 섹션 박스를 제어하는 Windows Forms 애플리케이션입니다. 이 애플리케이션은 사용자가 이동 간섭 검사를 진행할 수 있도록 합니다.

## 주요 기능
- 3D 모델 파일 열기
- 이동 간섭 검사
- 간섭 결과 확인

## 사용법
1. **모델 오픈** 버튼을 클릭하여 3D 모델(*.vizx) 파일을 엽니다.
2. **Group A 추가** 버튼을 클릭하여 A그룹을 지정합니다.
3. **Group B 추가** 버튼을 클릭하여 B그룹을 지정합니다. ( B그룹이 이동하며 검사됨. )
4. **충돌검사 추가** Add 버튼을 클릭하여 충돌검사를 추가합니다.
5. **이동경로 추가** 거리와 각도를 지정 후 Add 버튼을 클릭하여 Group B가 움직일 경로를 추가합니다.
6. **이동 간섭 검사 시작** Start 버튼을 클릭하여 이동 간섭 검사를 진행합니다.
7. **결과 확인** 결과가 종료되면 우측의 그리드 뷰로 결과 확인을 합니다.

## 파일 구조
- `FrmMain.cs`: 메인 폼 파일로, UI 이벤트 핸들러와 VIZCore3DX.NET 제어 로직을 포함합니다.
- `Program.cs`: 애플리케이션의 진입점입니다.

## 요구 사항
- .NET Framework 4.8
- VIZCore3DX.NET 라이브러리
